# API 一覧

VoiceBookLM バックエンド REST API の全体像です。

---

## 認証 API (`/api/auth`)

| メソッド | エンドポイント      | 説明                   | 認証 |
| :------: | ------------------- | ---------------------- | :--: |
|   POST   | `/api/auth/google`  | Google OAuth ログイン  | 不要 |
|   POST   | `/api/auth/refresh` | トークンリフレッシュ   | 不要 |
|   POST   | `/api/auth/logout`  | ログアウト             | 不要 |
|  DELETE  | `/api/auth/account` | アカウント削除         | 必須 |
|   GET    | `/api/auth/me`      | 現在のユーザー情報取得 | 必須 |

## 音声メモ API (`/api/voice`)

| メソッド | エンドポイント     | 説明                     | 認証 |
| :------: | ------------------ | ------------------------ | :--: |
|   POST   | `/api/voice/memos` | 音声ファイルからメモ生成 | 必須 |

## メモ API (`/api/memos`)

| メソッド | エンドポイント                | 説明         | 認証 |
| :------: | ----------------------------- | ------------ | :--: |
|   GET    | `/api/memos`                  | メモ一覧取得 | 必須 |
|   GET    | `/api/memos/{id}`             | メモ詳細取得 | 必須 |
|  PATCH   | `/api/memos/{id}`             | メモ更新     | 必須 |
|  DELETE  | `/api/memos/{id}`             | メモ削除     | 必須 |
|   POST   | `/api/memos/{id}/resummarize` | 再要約       | 必須 |

## フォルダー API (`/api/folders`)

| メソッド | エンドポイント      | 説明               | 認証 |
| :------: | ------------------- | ------------------ | :--: |
|   GET    | `/api/folders`      | フォルダー一覧取得 | 必須 |
|   POST   | `/api/folders`      | フォルダー作成     | 必須 |
|  PATCH   | `/api/folders/{id}` | フォルダー更新     | 必須 |
|  DELETE  | `/api/folders/{id}` | フォルダー削除     | 必須 |

## タグ API (`/api/tags`)

| メソッド | エンドポイント | 説明         | 認証 |
| :------: | -------------- | ------------ | :--: |
|   GET    | `/api/tags`    | タグ一覧取得 | 必須 |

## 開発用 API (`/api/dev`)

> [!WARNING]
> 開発環境（dev）でのみ有効。本番環境では使用不可。

| メソッド | エンドポイント   | 説明               | 認証 |
| :------: | ---------------- | ------------------ | :--: |
|   POST   | `/api/dev/token` | 開発用トークン取得 | 不要 |
|   POST   | `/api/dev/seed`  | テストデータ作成   | 必須 |
